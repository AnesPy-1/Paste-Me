# 📋 PasteMe | کلیپبورد آنلاین

**PasteMe** یک پلتفرم ساده و ایمن برای اشتراک‌گذاری سریع متن و فایل است. همانطور که از نام آن برمی‌آید، مثل کلیپبوردی آنلاین کار می‌کند—شما متن یا فایل خود را وارد می‌کنید تا یک کد منحصربه‌فرد دریافت کنید و با دیگران به اشتراک می‌گذارید.

---

## 🎯 پروژه چیست؟

تصور کنید که می‌خواهید یک فایل بزرگ یا متن طولانی را به دوست خود بدهید، اما نمی‌تواند سیستم‌های پیام معمول را کار کند یا وقت گذاشتن برای ایمیل ندارید. **PasteMe** دقیقاً برای این موارد ساخته شده است:

### 📝 متن؛ فایل:
- متن را کپی کنید و آن را در وب‌سایت جایگذاری کنید
- تا ۲۰۰ مگابایت فایل را آپلود کنید
- کد شش‌رقمی ساده دریافت کنید (مثلاً: ۱۲۳۴۵۶)
- این کد را به دیگران بدهید تا بتوانند متن یا فایل را دانلود کنند

### 🌍 عمومی و محلی:
- کاملاً متن‌باز و رایگان
- بدون نیاز به ثبت‌نام یا تسجیل‌ورود
- می‌تواند روی شبکه‌های خصوصی (بدون دسترسی به اینترنت) اجرا شود
- کاملاً فارسی‌زبان

### 🎨 تجربه‌ی کاربری:
- تم روشن و تاریک برای راحتی چشم در هر ساعتی
- طراحی واکنش‌پذیر برای کار بر موبایل، تبلت و رایانه
- سریع و بدون تاخیر در بارگذاری
- سایت‌ زیبا و مدرن

### 🔒 امنیت و کنترل:
- فایل‌ها فقط برای کسانی قابل دسترسی است که کد را بدانند
- می‌تواند فقط برای دامنه‌های خاص فعال شود (برای کسب‌وکار خصوصی)
- پنل مدیریتی محفوظ برای کنترل تنظیمات
- حمایت از لوگوی سفارشی و برندینگ

---

## 🌟 ویژگی‌های اصلی

✅ **اشتراک‌گذاری سریع:**
- کد ۶ رقمی تصادفی برای هر محتوا
- هیچ حساب یا کلمه‌عبور لازم نیست
- آپلود و دانلود سریع تا ۲۰۰ مگابایت

✅ **رابط فارسی و حرفه‌ای:**
- تمام متن‌ها به فارسی
- حالت RTL برای متن راست‌به‌چپ
- طراحی صحیح برای کاربران ایرانی

✅ **پنل مدیریتی:**
- کنترل کامل تنظیمات سایت
- قابل بسته‌شدن برای نگهداری یا بروزرسانی
- تغییر لوگو و برندینگ
- آمار استفاده و نظارت

✅ **سازگاری موبایل:**
- بدون تاخیر و فروپاشی در موبایل‌ها
- دکمه‌های بزرگ و قابل لمس
- سریع بارگذاری حتی روی شبکه‌های آهسته

✅ **سئو و موجودیت:**
- نقشه سایت خودکار برای موتورهای جست‌وجو
- فایل `robots.txt` برای کنترل ایندکس
- متاداده‌های مناسب برای به اشتراک‌گذاری در شبکه‌های اجتماعی

✅ **استقلال و انعطاف‌پذیری:**
- تمام فونت‌ها و استایل‌ها محلی (بدون وابستگی به سایت‌های خارجی)
- می‌تواند بدون اینترنت درون شبکه‌های خصوصی اجرا شود
- قابل سفارشی و تعدیل برای نیازهای خاص

## 🛠️ فناوری

**PasteMe** با فناوری‌های مدرن ساخته شده است:

- **Django** (فریم‌ورک وب Python): برای پرومپت و قابل اعتماد
- **SQLite**: برای ذخیره‌سازی ساده و بدون مسئله
- **HTML/CSS/JavaScript**: برای رابط زیبا و تعاملی
- **Vazirmatn Font**: درایت‌های فارسی حرفه‌ای
